================================================================================
                    AI VOICE AGENT - REQUIREMENTS
================================================================================

This document lists everything needed to run the AI Voice Agent project.

================================================================================
1. SYSTEM REQUIREMENTS
================================================================================

Operating System:
  - Windows 10/11, macOS, or Linux

Node.js:
  - Version: 18.0.0 or higher (recommended: 20.x LTS)
  - Download: https://nodejs.org/

npm:
  - Version: 9.0.0 or higher (comes with Node.js)

Browser (for client):
  - Chrome 90+ (recommended)
  - Firefox 90+
  - Edge 90+
  - Safari 15+
  Note: Must support WebRTC and Web Speech API

================================================================================

5. INSTALLATION STEPS
================================================================================

Step 1: Verify Node.js Installation
  $ node --version    # Should show v18.x.x or higher
  $ npm --version     # Should show 9.x.x or higher

Step 2: Clone/Copy the Project
  Copy the project folder to your desired location
  (Do NOT copy node_modules folders - they will be installed fresh)

Step 3: Install All Dependencies
  $ cd voiceagent-Copy
  $ npm run install:all

Step 4: Configure Environment Variables
  $ copy .env.example .env     # Windows
  $ cp .env.example .env       # Linux/Mac
  
  Then edit .env and add your API keys:
    DEEPGRAM_API_KEY=your_deepgram_api_key
    GROQ_API_KEY=your_groq_api_key

Step 5: Run the Project
  $ npm run dev

Step 6: Open in Browser
  Navigate to: http://localhost:3000

7. TROUBLESHOOTING
================================================================================

Problem: "ETIMEDOUT" or "Connection error" to Groq API
Solution: The project is configured to use IPv4. If issues persist, check
          your internet connection or try using a mobile hotspot.

Problem: "EADDRINUSE" error
Solution: Kill existing Node.js processes:
          $ Get-Process -Name "node" | Stop-Process -Force  (Windows PowerShell)
          $ killall node                                     (Linux/Mac)

Problem: Microphone not working
Solution: Allow microphone access in browser when prompted.
          Check browser settings for microphone permissions.

Problem: No voice output
Solution: Check browser audio settings. The Web Speech API requires
          the page to be interacted with before playing audio.
